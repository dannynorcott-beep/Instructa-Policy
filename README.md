Instructa Privacy Policy:
Last updated: 14 April 2026
Instructa is a sailing instructor planning and debrief tool. This Privacy Policy explains what data the app stores, how it is used, and what happens when a user chooses to export or send backup data for support or feedback.

Who we are:
Instructa is provided by Danny Norcott.
Contact: dannynorcott@gmail.com

What data Instructa stores:
Instructa may store data entered by the user, including:
week and course setup information
student first names
course assignments
roster notes and student notes
planned and delivered training activities
observation notes
competency coverage and confirmation records
app settings such as the last viewed week

Recommended user practice:
Users should:
use student first names only
keep notes professional, relevant, and training-focused
avoid entering unnecessary sensitive personal information

How the data is used:
The data stored in Instructa is used to support:
course planning
daily training delivery
student debriefing
competency tracking
continuity between training days and weeks
product support and beta feedback when the user chooses to send a backup

Backup exports and feedback emails:
Instructa allows the user to export a backup of app data and, in some cases, attach that backup to an email for feedback or support.
A backup may include:
student first names
notes entered by the instructor
observations
week and course structure
activity history
competency and progress data
Backups are only exported, shared, or emailed when triggered by the user.
Data exported from the app is handled via third-party services (e.g. email providers), which may apply their own security and encryption standards.

Data sharing:
Instructa does not automatically send user data to the developer from within the app.
Data is only shared externally when the user chooses to:
export a backup
share a backup
send a feedback or support email with a backup attached
When a user sends a backup by email, that data is shared with the selected email provider and the support recipient.

Data storage:
Instructa stores data locally on the user’s device and may create exported backup files in device-accessible storage or app documents storage, depending on device capabilities.
Data retention and deletion
Data remains stored until the user deletes it, uninstalls the app, or removes exported backup files.
If a user emails backup data for support or feedback, that email and attachment may be retained for a reasonable period to investigate issues, improve the product, and support beta testing.

Data Deletion
Instructa stores data locally on your device.
If you have shared data with us (for example via email backup), you can request deletion by contacting:

Instructa.app@gmail.com

We will delete any data we hold upon request.

Security:
Instructa aims to handle stored data using standard platform storage and sharing mechanisms. Users should avoid storing unnecessary sensitive personal data in the app.

Changes to this policy:
This Privacy Policy may be updated from time to time. The latest version will be published at the policy URL provided in the app and on Google Play.
